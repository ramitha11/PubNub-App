Reserve is your personal dining concierge. With just a few taps, Reserve enables a more seamless experience, by dining without waiting. Enter your choice to get information on your fingertips.

How to run Reserve

To run Reserve, launch the user.html webpage. This web page incorporates PubNub publish() and subscribe() calls using client architecture. The user publishes the information on a certain channel to which all the restaurants subscribe to using PubNub datastream applications. Similarly the restaurants publish on a different channel to which the user has subscribed to. Restaurant of choice is entered by the user which redirects him to a webpage that displays the availablity of vacant tables for that restaurant.
